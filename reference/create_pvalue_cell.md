# Create P-value Cell Optimized

Creates optimized p-value computation cell.

## Usage

``` r
create_pvalue_cell(var_name, grp_var, test_type, data = NULL)
```

## Arguments

- var_name:

  Variable name

- grp_var:

  Grouping variable name

- test_type:

  Type of statistical test

## Value

P-value cell object
