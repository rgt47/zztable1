# Populate Variable Cells Optimized

Dispatcher function for efficient population of variable cells. Routes
to stratified or non-stratified implementation based on analysis.

## Usage

``` r
populate_variable_cells(blueprint, data, dimensions, theme_config)
```

## Arguments

- blueprint:

  Blueprint object

- data:

  Data frame

- dimensions:

  Dimension analysis

- theme_config:

  Theme configuration

## Value

Blueprint with variable cells populated
