# Populate Variable Cells - Simple (Non-Stratified) Analysis

Handles variable cell population for non-stratified tables. Simpler
logic without stratum handling.

## Usage

``` r
populate_variable_cells_simple(blueprint, data, dimensions, theme_config)
```

## Arguments

- blueprint:

  Blueprint object

- data:

  Data frame

- dimensions:

  Dimension analysis

- theme_config:

  Theme configuration

## Value

Blueprint with cells populated
