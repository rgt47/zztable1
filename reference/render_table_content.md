# Render table content based on output format

Optimized rendering logic that iterates over existing cells rather than
all possible grid positions. This is much more efficient for the sparse
tables generated by this package.

## Usage

``` r
render_table_content(blueprint, theme, format)

render_table_content(blueprint, theme, format)
```

## Arguments

- blueprint:

  Table1Blueprint object

- theme:

  Theme configuration

- format:

  Output format (console, latex, html)

## Value

Character vector with formatted table content

Character vector with table content
