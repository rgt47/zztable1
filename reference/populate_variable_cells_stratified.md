# Populate Variable Cells - Stratified Analysis

Handles variable cell population for stratified tables. Each stratum
gets its own section with header and variable data.

## Usage

``` r
populate_variable_cells_stratified(blueprint, data, dimensions, theme_config)
```

## Arguments

- blueprint:

  Blueprint object

- data:

  Data frame

- dimensions:

  Dimension analysis

- theme_config:

  Theme configuration

## Value

Blueprint with stratified cells populated
