# Populate Numeric Variable Optimized

Efficient population of numeric variable cells.

## Usage

``` r
populate_numeric_variable(
  blueprint,
  var_name,
  data,
  start_row,
  dimensions,
  theme_digits,
  theme_name
)
```

## Arguments

- blueprint:

  Blueprint object

- var_name:

  Variable name

- data:

  Data frame

- start_row:

  Starting row

- dimensions:

  Dimension analysis

- theme_digits:

  Theme decimal places

## Value

Blueprint with numeric cells populated
