# Format Summary Row Values Into Output Lines

Format Summary Row Values Into Output Lines

## Usage

``` r
format_summary_lines(summary_vals, format, theme, type = "group")
```

## Arguments

- summary_vals:

  List of named character vectors from evaluate_summary_for_group

- format:

  Output format

- theme:

  Theme configuration

- type:

  "group" or "grand"

## Value

Character vector of formatted summary lines
