# Populate Binary Variable (Collapsed)

For 2-level factors, renders as a single row showing the count and
percentage of the second (positive) level. The variable label includes
'– no. (

## Usage

``` r
populate_binary_variable(
  blueprint,
  var_name,
  data,
  start_row,
  dimensions,
  theme_digits
)
```

## Arguments

- blueprint:

  Blueprint object

- var_name:

  Variable name

- data:

  Data frame

- start_row:

  Starting row

- dimensions:

  Dimension analysis

- theme_digits:

  Theme decimal places

## Value

Blueprint with binary variable cells populated
