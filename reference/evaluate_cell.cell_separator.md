# Evaluate Separator Cell

S3 method for evaluating separator cells.

## Usage

``` r
# S3 method for class 'cell_separator'
evaluate_cell(
  cell,
  data,
  env = parent.frame(),
  force_recalc = FALSE,
  blueprint = NULL
)
```

## Arguments

- cell:

  Separator cell object

- data:

  Data frame (not used)

- env:

  Evaluation environment (not used)

- force_recalc:

  Force recalculation (not used)

- blueprint:

  Blueprint object (optional)

## Value

Separator string
