# Populate Factor Variable Optimized

Efficient population of factor variable cells.

## Usage

``` r
populate_factor_variable(
  blueprint,
  var_name,
  data,
  start_row,
  dimensions,
  theme_digits
)
```

## Arguments

- blueprint:

  Blueprint object

- var_name:

  Variable name

- data:

  Data frame

- start_row:

  Starting row

- dimensions:

  Dimension analysis

- theme_digits:

  Theme decimal places

## Value

Blueprint with factor cells populated
