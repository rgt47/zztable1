# Get theme metadata

Retrieves metadata about a registered theme.

## Usage

``` r
get_theme_metadata(theme_name)
```

## Arguments

- theme_name:

  Character string with theme name

## Value

List with theme metadata (name, author, version, description)

## Examples

``` r
if (FALSE) { # \dontrun{
get_theme_metadata("nejm")
} # }
```
