# Apply Footnote Marker Optimized

Apply Footnote Marker Optimized

## Usage

``` r
apply_footnote_marker(
  text,
  marker_key,
  footnote_markers,
  layout,
  style = "numbers"
)
```

## Arguments

- text:

  Text to potentially mark

- marker_key:

  Marker key to check

- footnote_markers:

  List of footnote markers

- layout:

  Output layout

## Value

Text with marker applied if applicable
