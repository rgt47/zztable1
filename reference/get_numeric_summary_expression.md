# Get Numeric Summary Expression Optimized

Get Numeric Summary Expression Optimized

## Usage

``` r
get_numeric_summary_expression(summary_type, digits = 2, theme_name = NULL)
```

## Arguments

- summary_type:

  Summary type specification

- digits:

  Number of decimal places

- theme_name:

  Theme name for format selection (NEJM uses ± format)

## Value

Expression for numeric summary
