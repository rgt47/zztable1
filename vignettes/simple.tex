\section{Introduction}\label{introduction}

This vignette demonstrates the versatility of \texttt{zztable1\_nextgen}
using various built-in R datasets. We'll explore different argument
combinations and show how the package handles different types of
variables and data structures commonly encountered in statistical
analysis.
