\section{Introduction}\label{introduction}

This comprehensive vignette showcases \textbf{all available themes} in
\texttt{zztable1\_nextgen} using carefully selected built-in R datasets.
Each theme is designed to match specific publication standards, from
medical journals to general statistical reports.

\subsection{Available Themes}\label{available-themes}

The package includes \textbf{5} built-in themes:

\begin{table}
\centering
\caption{\label{tab:theme-list}Available Themes in zztable1nextgen}
\centering
\begin{tabular}[t]{l|l}
\hline
Theme & Description\\
\hline
console & Console - Basic monospace output for development\\
\hline
nejm & NEJM - New England Journal of Medicine styling with authentic cream striping\\
\hline
lancet & Lancet - Clean minimal formatting matching The Lancet\\
\hline
jama & JAMA - Journal of American Medical Association styling\\
\hline
simple & Simple - Clean general-purpose theme for reports\\
\hline
\end{tabular}
\end{table}

Each theme will be demonstrated using the same dataset to clearly show
the formatting differences.

\section{Conclusion}\label{conclusion}

The \texttt{zztable1\_nextgen} package provides a flexible and efficient
way to create publication-ready ``Table 1'' summaries. The examples in
this vignette demonstrate:

\begin{itemize}
\tightlist
\item
  \textbf{Parameter Flexibility}: \texttt{strata}, \texttt{missing},
  \texttt{pvalue}, \texttt{totals}, and \texttt{footnotes} parameters
\item
  \textbf{Theme Variety}: All 5 built-in themes with authentic journal
  formatting
\item
  \textbf{Footnote Support}: Both numbered (NEJM, Simple) and lettered
  (JAMA, Lancet) footnote styles
\item
  \textbf{Missing Data Handling}: Comprehensive missing value reporting
  when \texttt{missing=TRUE}
\item
  \textbf{Stratified Analysis}: Multi-group comparisons using the
  \texttt{strata} parameter
\item
  \textbf{Performance}: Efficient handling of large datasets with
  complex parameter combinations
\end{itemize}

Key footnote features demonstrated: - \textbf{NEJM Theme}: Numbered
footnotes (1, 2, 3) for clinical publications - \textbf{JAMA Theme}:
Lettered footnotes (a, b, c) for medical research\\
- \textbf{Simple Theme}: Numbered footnotes for general reports -
\textbf{Custom Content}: Flexible footnote text for methods, data
sources, and definitions

The package maintains the familiar R formula interface while providing
significant performance improvements and enhanced functionality through
its optimized architecture.
