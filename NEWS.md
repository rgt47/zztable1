# zztable1 0.1.0

## Initial Release

### Core Features

* Formula-based interface for summary table specification
* Lazy evaluation blueprint architecture for efficient table generation
* Sparse storage implementation using R environments
* Multi-format output support (console, HTML, LaTeX)

### Medical Journal Themes

* NEJM theme with authentic styling (light yellow striping, minimal borders)
* Lancet theme (clean white background, horizontal borders)
* JAMA theme (minimal formatting, lettered footnotes)
* Console theme for development and testing

### Statistical Features

* Configurable statistical tests for continuous and categorical variables
* Support for t-test, ANOVA, Welch, Kruskal-Wallis tests
* Fisher's exact and chi-squared tests for categorical data
* Flexible numeric summary options (mean_sd, median_iqr, mean_ci)

### Advanced Capabilities

* Stratified analysis support for multi-center trials
* Missing data handling with per-variable missing rows
* Custom summary function support
* Comprehensive footnote system

### Documentation

* Complete vignette system with examples
* Theming system documentation
* Stratified analysis examples
* Dataset examples using built-in R data
